Vogel's Approximation Method

VAM (Vogel's Approximation Method) is an iterative process for determining the first viable solution to a transportation problem. Like the Least Cost Method, the delivery cost is included here as well, but only in a relative sense.

Because the initial basic viable solution provided by this approach is either ideal or extremely close to the optimal solution, it is chosen above the NWCM and LCM.

Steps for Vogel’s approximation method:

  1. Start with calculating the difference of small cost and next higher to the smallest cost for each row and column. These differences are called the row and column penalties respectively
  2. Take the difference of two smallest numbers in each raw and write it in the row penalty
  3. Take the difference of the two smallest numbers in each column and write it in the column penalty
  4. Select the highest penalty and observe the corresponding row and column then allocate min (Ai, Bj) in the cell having the lowest cost in the select row and column.
  5. Ignored satisfied row or column for further consideration. Calculate fresh penalties for remaining sub metrics (m) in the same manner
  6. Repeat the steps until all Supply and Demand values are 0.

(Note that the null row is added to make supply and demand values equal.)
